# Department Management System (DMS)

## Project Objective

Build a complete, production-quality **Department Management System
(DMS)** using **PHP 8.3 + MySQL + PDO** for XAMPP. This is a real
software project, not a demo or code snippets.

## Core Requirements

-   One continuous codebase.
-   Never regenerate previous modules; extend the existing project.
-   Final deliverable: complete ZIP with source, SQL, assets,
    documentation.

## Technology

-   PHP 8.3
-   MySQL
-   PDO
-   HTML5
-   CSS3
-   Vanilla JavaScript
-   Chart.js
-   DomPDF
-   Remita Payment Gateway

## UI / UX

-   Inspired by Notion, Linear and Vercel.
-   Minimalist, premium, handcrafted.
-   No Bootstrap templates.
-   No shadows.
-   No gradients.
-   No glassmorphism.
-   Light typography, bold headings only.
-   Responsive.
-   CSS variable based Light/Dark mode.

### Authentication

Roles: - Admin - Staff - Student

Features: - Register - Login - Logout - Forgot/Reset Password - Password
hashing - Sessions - RBAC

Students and Staff self-register. Admins can activate, deactivate,
suspend and delete users.
